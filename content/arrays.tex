\chapter{Arrays}

Basics taken from topskip on SO (https://stackoverflow.com/questions/2647674/does-latex-have-an-array-data-structure).
Array implementation expands on this in a more generic way.

\section{Declaration}
\newcounter{mycounter}
\setcounter{mycounter}{1}
\expandafter\newcommand\csname myarr\the\value{mycounter} \endcsname{myfirstelement}
\stepcounter{mycounter}
\expandafter\newcommand\csname myarr\the\value{mycounter} \endcsname{mysecondelement}

\section{Access hardcoded}
\csname myarr1 \endcsname

\section{Access using variable}
\newcounter{index}
\setcounter{index}{2}
\csname myarr\the\value{index} \endcsname

\section{Stack}

\arrCreate{stack}
\arrPushVal{stack}{first item}
\arrPushVal{stack}{second item}
\arrPushVal{stack}{third item}

\subsection{Looping over it}
\arrPrint{stack}
